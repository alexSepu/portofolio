How has it been done ?

      1. Structure

      This website is a single page website that is developed with HTML and Botstrap to update the web disposition.

      2. Styles

      To design the website use CSS with some animation, for example, to rotate the logo. I have also used Google Font when importing some fonts to use in titles.

      3. Hosting

      We can find the hosting on Netlify, which is a website that uses Github repositories to submit files.

      4. Multilanguage

      To configure the multilanguage I used JavaScript to configure a cookie with the language and two JSON files to save all the translations in Spanish and English.

      5. Icons

      For example, I have used Font awesome, which is a library that provides many icons that you can change in width or color. On this website I use it to complement the navigation bar, some skills, ...
